# 关于 aar for vscode

这是一个在 vscode 中编写和运行 aardio 代码的插件，以获得良好的体验。

## 功能

[ ] 运行整个文件或所选内容。
[ ] 语法高亮。支持关键字、函数、甚至嵌套语言。
[ ] 自动补全。补全过程也显示了相关使用方式。
[ ] 发布。读取工程信息并发布。
[ ] 代码片段。常用而快捷的代码片段支持。
[ ] 实时文档。鼠标在某个关键字上时显示简略释意，点击更多查看相关文档。
[ ] 快捷键支持。
[ ] 更多源 ide 的实现...

## 插件设置
为了模仿接近源 ide 的运行环境， 使用前请设置你的 aardio ide 路径。

## 已知的问题
