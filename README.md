# 关于 aar-vsc
在 vscode 中编写和运行 aardio 代码.  
优先保证基本的编码体验, 然后尝试添加其他功能.  

## 功能
- 运行文件(ctrl+r)
- 运行所选内容(ctrl+r)
- 语法高亮
- 自动补全
- 补全过程中显示函数使用方法
- 代码片段, 常用而快捷的代码片段支持

## 提示
插件默认的 aardio 运行环境为 v26.5,  
你可以设置你的 aardio ide 路径, 以获得接近 ide 的运行环境.  


## 已知的问题
